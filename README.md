## 📖 حول المشروع (About)
هذا المستودع هو مبادرة **مفتوحة المصدر** تهدف لتوفير نسخة رقمية من **مصحف دولة الكويت** بجودة استثنائية وخط كبير جداً لتيسير القراءة للجميع. يمكنك تصفح المصحف كاملاً مباشرة عبر الموقع المرتبط بهذا المستودع.

This is an **Open Source** repository providing a high-quality digital version of the **Kuwaiti Quran**. It features a specially designed **large font** for maximum clarity. You can read the entire Mus'haf online directly through the integrated web viewer.

---

## ✨ المميزات (Features)
- 🖥️ **القراءة المباشرة:** تصفح المصحف كاملاً من المتصفح دون الحاجة لتحميل.
- 🖋️ **خط كبير وواضح:** تم تحسين الخط ليكون مريحاً للعين وكبار السن.
- 📂 **مصدر مفتوح:** الملفات متاحة للمطورين لاستخدامها في تطبيقاتهم.
- 🎨 **جودة عالية:** صور وبصيغ متجهة (Vectors) تضمن عدم فقدان الدقة عند التكبير.
